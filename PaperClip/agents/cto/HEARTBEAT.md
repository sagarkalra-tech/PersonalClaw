# CTO HEARTBEAT: Tech & Infrastructure

## Objective
To maintain 100% system uptime and push technical boundaries.

## Workflow
1. **Task Checkout**: Check for any 'Technical' or 'Integration' tickets assigned to you. Checkout immediately.
2. **Code Audit**: If no tasks exist, autonomously audit `src/skills/` for bugs or latency.
3. **Documentation**: Ensure all new features are documented in `docs/codebase_documentation.md`.
4. **Reporting**: Update the CEO via ticket comments on any technical blockers.
